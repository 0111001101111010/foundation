--- title: Media Object layout: doc/layouts/default.html --- # Media Object

The media object is a standard object type built for convenience (and as a Flexbox demo, frankly). It's useful for any instance of a piece of media (image, icon, etc) next to some content like text.

***

"There is no belief, however foolish, that will not gather its faithful adherents who will defend it to the death." — Isaac Asimov

***

Basic

A media object is made of 3 components: a media-object container, a media element, and the content element. Foundation is not opinionated about the tags you use, only the classes, so any standard block or inline (they will be made block level by flexbox) will work.

Advanced

By default, on mobile (small) devices the media element will be centered above the content. However you can have them side-by-side in the more traditional form by adding a class of .persist to the media object.

You can flip the media object on larger devices from media left, content right to media right, content level by adding a class of .reverse to the object. This will also flip on small devices from media top, content bottom to content top, media bottom.
